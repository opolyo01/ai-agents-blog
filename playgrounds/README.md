# Playgrounds

Scratch areas for experiments that should not live inside the published blog app.

## Purpose

Use `playgrounds/` for:

- framework spikes
- rough scripts
- prototype integrations
- learning notes that may later become real posts

## Current labs

- `langgraph-lab/`
  A focused sandbox for learning LangGraph and LangChain by building tiny examples first.

## Working rule

If the code is exploratory, noisy, or likely to be deleted, it belongs here first.
